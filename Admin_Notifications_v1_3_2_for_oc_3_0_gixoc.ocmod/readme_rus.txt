Модуль уведомлений администраторов магазина о новом:
1) заказе
2) покупателе
3) партнере (убрано в версии для Opencart 3.0, т.к. личный кабинет покупателя и личный кабинет партнера в этой версии Opencart'a были объединены) 
4) отзыве
5) возврате


Возможности:
1. Гибкая, мультиязычная настройка шаблона уведомления.
2. Гибкая настройка отправки уведомлений для каждого администратора - например, если нужно уведомлять только одного администратора об возвратах, установите только у него "галочку".
3. Возможность настройки уведомления для каждого из статусов заказа.
4. Возможность установки логгирования ошибок запросов (рекомендуется включать только на этапе тестирования).
5. Настройка действия при длинном сообщении (обрезать или разбивать на несколько). Добавлено в связи с ограничением длины символов в одном сообщении.
6. Возможность установки таймаута соединения (времени в секундах подключения к серверам). Добавлено для предотвращения длительного "зависания" во время оформления заказа, регистрации и тд в случае недоступности серверов месседжеров.
7. Поддержка прокси (Socks 5) для месседжера Telegram.
8. Подробная инструкция по настройке модуля.
9. Поддержка полей модуля Simple
10. Возможность отправки уведомлений не только администраторам, но и другим пользователям. Для месседжера Telegram также добавлена возможность отправки уведомлений в группу.


Распространяется: "как есть" (as is).


Версия Opencart (OcStore): 3.0


Техподдержка в свободное время
